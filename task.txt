1️⃣ Input (Data the user enters)

When adding a book, the user must give information about the book.

Example input:

Enter Book ID: 101
Enter Book Name: CleanCode
Enter Author Name: RobertMartin
Enter Quantity: 3

So the inputs are:

Book ID

Book Name

Author Name

Quantity

These will be stored inside a Book object.

Example class:

class Book
{
public:
    int id;
    string name;
    string author;
    int quantity;
};
2️⃣ Output (What the program shows)

Output is the information printed on screen.

Example:

----- Book List -----

ID: 101
Name: CleanCode
Author: RobertMartin
Available: 3

So the program reads the stored books and prints them.
